# Лабораторна робота 4: API для прогнозування ожиріння

Цей проект є FastAPI-додатком, який надає API для тренування та прогнозування рівня ожиріння на основі моделі XGBoost, навченої в Лабораторній 3.

## 1. Встановлення та підготовка

1.  **Створіть базу даних:**
    Скопіюйте файл `task-d.csv` з папки `src/` у корінь проєкту (де цей `README.md`).
    Запустіть скрипт `src/ImportToDb.py`, щоб створити `lab3.db` та наповнити його даними.

2.  **Активуйте віртуальне середовище:**
    ```powershell
    .\.venv\Scripts\Activate.ps1
    ```

3.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```

## 2. Запуск сервера

Для запуску сервера FastAPI у режимі розробки (з автоматичним перезавантаженням) виконайте з кореневої папки:

```bash
uvicorn app.main:app --reload
