# Лабораторна робота 4: API для прогнозування ожиріння

Цей проект є FastAPI-додатком, який надає API для тренування та прогнозування рівня ожиріння на основі моделі XGBoost, навченої в Лабораторній 3.

## Встановлення

1.  Встановіть залежності:
    ```bash
    pip install -r requirements.txt
    ```
2.  Переконайтеся, що у вас є файл `lab3.db` з таблицею `obesity_data` в кореневій папці проекту.

## Запуск сервера

Для запуску сервера FastAPI у режимі розробки (з автоматичним перезавантаженням) виконайте:

```bash
uvicorn app.main:app --reload
